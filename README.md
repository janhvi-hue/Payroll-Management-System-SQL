# Payroll Management System (SQL Project)

## 📌 Overview
This project is a SQL-based Payroll Management System designed to manage employee, department, and salary data efficiently.

## 🛠️ Tools & Technologies
- MySQL
- SQL
- DBMS

## 📂 Database Structure
- Departments Table
- Employees Table
- Payroll Table

## ⚙️ Features
- Employee and department management
- Salary calculation using allowances and deductions
- Net salary view creation
- Data analysis using aggregate functions
- Employee classification using CASE statements

## 📊 Key Concepts Used
- Joins (INNER, LEFT)
- Aggregate Functions (SUM, AVG, COUNT, MAX, MIN)
- GROUP BY & HAVING
- Views
- CASE Statements

## 🚀 Sample Queries
- Department-wise salary analysis
- Net salary calculation
- Employee categorization
